1- برای استفاده از گیت ، گیت را نصب کنید
https://git-scm.com/

2- برای استفاده راحت از گیت GitKraken نصب 
https://www.techspot.com/downloads/downloadnow/7074/?evp=da25f7b72b5b90222fcfbdcd9b1c129a&file=9279

3- پروژه خود را ایجاد کنید git bash بعد از نصب گیت و ثبت نام در گیت کراکن با زدن کلیک راست و باز کردن  
mkdir projectname

3.5- رفتن به داخل پوشه ی پروژه
cd projectname

4- دستور اضافه کردن گیت به پروژه
git init

5- Vscode از GitLens نصب اکستنشن

6- میشود هم کامیت و هم استیج و هم پوش کرد نیازی به دستورات کامند لاین نیست vscode در 

7- باز کردن پروژه در گیت کراکن

8- ساخت ریپازیتوری در گیت هاب برای پروژه

9- ریموت کردن پروژه لوکال روی گیت هاب با گیت کراکن

10- می شود فایل هایی که تغییر می کنند و نیازی به کامیت ندارن رو مشخص کنید میتونید از سایت زیر هم استفاده کنید .gitignore با استفاده از فایل
gitignore.io

11- Branch = ورژن جدید

12- Pull = اتصال برنج ها به گیت هاب

13- Push = اضافه کردن تغییرات پروژه لوکال به گیت هاب

14- Stash = تغییراتی که کامیت نشدن و میخواهم در تاریخچه گیت حذف نشن

15- Pop = برداشتن از حالت استش

16- شما میتوانید با استفاده از گیت کراکن و کامند لاین برنچ و پوش و پل و استش و پاپ کنید

17- اچ تی تی پی اس پروژه را کپی کرد clone برای دسترسی اعضای گروه به پروژه گروهی اول باید به ریپازیتوری پروژه در گیت هاب رفت و در قسمت کد قسمت 

18- سپس دوباره گیت بش را باز کنید و برای گرفتن پروژه دستور زیر را وارد کنید
git clone <url>

19- پروژه را در وی اس کد و گیت کراکن باز کنید

20- و از پروژه استفاده کنید
